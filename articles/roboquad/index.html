Roboquad: Quest to Hard Mod

11/21/2024

Project

I purchased a WowWee Roboquad, a cute robot toy from the early 2000's. This was purchased for the sole reason of, I wanted to see if I could modify the hardware to make it a bit more interesting. For this project I plan to design some fully custom PCBs, as well as modify the existing ones. This is currently ongoing, and I'm learning about the machine in the meantime, but will be adding updates and notes as I learn more about this toy.
