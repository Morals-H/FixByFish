Information Collected

I only intentionally collect minimal, non-personal information, such as the type of browser, operating system, and approximate location of the connection. I do not use analytics from any corporation I don't trust, so rest assured no Google. My analytics tool of choice is GoatCounter, an absolutely awesome FOSS (Free & Open Source software) project.

If you'd prefer to share as little information as possible try uBlock Origin. This extension can prevent the analytics request entirely. If you're otherwise uncomfortable with this, feel free to contact me.
